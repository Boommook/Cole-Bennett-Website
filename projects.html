Attack Lab

Implemented code injection and return-oriented programming attacks for Attack Lab. This lab was an assigment for Machine Organization and Assembly Language or CS 2011 at WPI. To assist in this lab, I used GDB to walk through the code. Attack Lab required a strong understanding of the x86-64 assembly language. View Attack Lab posted by CU Boulder by clicking the button below. Above are two of the four phases that I completed, the left is phase 2, a demonstrating a code injection attack, while the right is phase 4, which has the same goal as phase 2, but instead uses a return-oriented programming attack. By viewing the images in full screen, you can better view the explanations between each line of my work.

The Other Side

The Other Side is a strategic competitive board game to be played with 4 friends! This game was created by me and 3 others for Digital Game Design I. We were limited to making the rules had to fit on the front and back of a page and must be able to be played with common school or home items. Making this game helped me develop my game development skills.
